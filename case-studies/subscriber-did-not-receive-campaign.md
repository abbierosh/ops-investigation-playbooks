# Case Study: Subscriber Did Not Receive Campaign

## Issue

A team wanted to know why one subscriber did not receive a campaign.

## Checked

1. Subscriber was valid and contactable
2. Interaction history
3. Target audience membership
4. Send, bounce, unsubscribe, or exclusion events
5. Whether this was isolated or wider

## Read

This separates:

- not sent
- sent but not noticed
- valid exclusion
- wider workflow or data issue
