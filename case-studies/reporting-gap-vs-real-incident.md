# Case Study: Reporting Gap vs Real Incident

## Issue

A campaign looked underdelivered soon after launch.

## Checked

1. Campaign status and activation time
2. Expected audience vs send count
3. First and latest send events
4. Whether send counts were still moving
5. Whether other campaigns showed the same gap

## Read

Send events were still moving, so I would not treat it as a failed send yet.

The issue was early reporting looking more final than the event data really was.
